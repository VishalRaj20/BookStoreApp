# BookStoreApp

An online platform to browse, read, and access books across various genres like Fiction, Non-Fiction, Tech, and more.

🔹 Features
✅ Search & explore books
✅ Read previews before purchase
✅ Access free & premium books
✅ User-friendly interface

🔹 Installation
git clone https://github.com/VishalRaj20/BookStoreApp.git
cd BookStore
npm install
npm start
